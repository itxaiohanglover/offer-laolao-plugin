# 🚀 Offer Laolao - Интеллектуальный плагин для браузера для автоматического заполнения резюме

> Мощный расширение для браузера Chrome, поддерживающее двойной режим интеллектуального анализа резюме и ручного заполнения, с функциями **интеллектуального сопоставления полей с помощью ИИ** и **точного заполнения на уровне полей**, помогающее соискателям быстро и точно заполнять резюме на крупных сайтах по поиску работы.

![Version](https://img.shields.io/badge/Version-1.0-blue)
![Chrome](https://img.shields.io/badge/Chrome-Extension-green)
![Manifest](https://img.shields.io/badge/Manifest-V3-orange)

🌐 **Онлайн-доступ**：[https://offer-laolao-plugin.vercel.app](https://offer-laolao-plugin.vercel.app)

## ✨ Основные функциональные характеристики

### 📄 Интеллектуальный анализ резюме

- **Поддержка нескольких форматов**：Поддерживает основные форматы резюме, такие как PDF, DOCX, DOC, TXT, JSON
- **Анализ через облачный API**：Интегрирует API анализа резюме с рынка Alibaba Cloud для точной экстракции информации из резюме
- **Прямой импорт JSON**：Поддерживает прямой импорт данных резюме в формате JSON для резервного копирования и восстановления данных
- **Загрузка через перетаскивание**：Поддерживает загрузку файлов через перетаскивание для удобной работы
- **Интеллектуальное сопоставление полей**：Автоматически сопоставляет результаты анализа с соответствующими полями формы

### 📝 Полное управление информацией о резюме

Поддерживает заполнение и управление следующими модулями резюме：

| Модуль              | Включенные поля                                                                     |
| ----------------- | ---------------------------------------------------------------------------- |
| **Основная информация**      | Имя, Пол, Дата рождения, Номер мобильного телефона, Email, Номер паспорта, Место жительства, Политическая ориентация         |
| **Желаемая работа**      | Желаемая должность, Желаемая отрасль, Желаемая зарплата, Желаемое место работы, Продолжительность стажировки, Возможное время работы                 |
| **Образование**      | Название школы, Специальность, Уровень образования (специалитет/бакалавр/мастер/доктор), Рейтинг, Даты поступления/выпуска (поддержка нескольких записей) |
| **Опыт работы/стажировки** | Название компании, Должность, Даты начала/окончания, Описание работы (поддержка нескольких записей)                          |
| **Проектный опыт**      | Название проекта, Роль, Срок проекта, Описание проекта, Описание обязанностей (поддержка нескольких записей)                 |
| **Информация о навыках**      | Название навыка, Уровень навыка (начальный/средний/продвинутый/эксперт) (поддержка нескольких записей)                        |
| **Языковые навыки**      | Название языка, Уровень владения (начальный/базовый/продвинутый/идеальный), Языковые сертификаты (поддержка нескольких записей)              |
| **Пользовательские поля**    | Названия и содержание пользовательских полей (поддержка нескольких записей)                                             |
| **Самопрезентация**      | Демонстрация личных сильных сторон и особенностей                                                           |

### 🎯 Интеллектуальное заполнение форм

#### Функция предзаполнения одним щелчком

- Нажмите кнопку "📋 Предзаполнить", чтобы автоматически заполнить данные резюме в форму текущего сайта по поиску работы
- Интеллектуально распознает поля формы, автоматически сопоставляет соответствующую информацию из резюме
- Поддерживает различные элементы формы, такие как input, textarea, select, contenteditable
- Автоматически запускает события формы (input, change, blur), чтобы гарантировать прохождение проверки сайта
- После заполнения предоставляет визуальную обратную связь, выделяя заполненные поля

#### Точное заполнение на уровне полей（↗ Заполнение указанием）

- Каждый поле имеет кнопку "↗" для **точного заполнения одного поля**
- После нажатия кнопки входит в "режим заполнения указанием"：
  - В верхней части страницы отображается подсказка по操作
  - Когда мышь наводится на заполняемый элемент, он выделяется (синий бордер)
  - Нажмите на целевой ввод, чтобы ввести значение поля
  - Нажмите клавишу `Esc`, чтобы отменить操作
- Поддерживает заполнение в элементы input, textarea, select, contenteditable
- Автоматически запускает события формы (input, change, blur), чтобы гарантировать прохождение проверки сайта
- После успешного заполнения автоматически закрывает всплывающее окно, удобно для последовательных操作

### 🤖 Интеграция с большими моделями ИИ

Поддерживает несколько китайских провайдеров больших моделей для интеллектуальной оптимизации содержимого резюме и сопоставления полей：

| Провайдер услуг                | Поддерживаемые модели                                               |
| --------------------- | ------------------------------------------------------ |
| **DeepSeek**          | DeepSeek Chat、DeepSeek Coder                          |
| **Kimi (Moonshot)**   | Moonshot 8K/32K/128K                                   |
| **Tongyi Qianwen (Alibaba Cloud)** | Qwen Turbo/Plus/Max/Max Long Text                         |
| **Volcano Engine (Doubao)**   | Doubao Seed 1.6、Doubao Seed 1.6 Lite、Doubao Seed 1.6 Flash |
| **Zhipu AI**           | GLM-4、GLM-4 Flash、GLM-3 Turbo                        |
| **Baichuan Intelligence**          | Baichuan 2 Turbo、Baichuan 2 Turbo 192K                        |
| **Пользовательский**            | Поддерживает любой API совместимого с OpenAI формата                         |

**Функциональные характеристики ИИ**：

- Проверка подключения API одним щелчком
- **✨ Оптимизация резюме с помощью ИИ одним щелчком**：Интеллектуально оптимизирует самоопределение, описание работы, описание проекта и т.д.
- **🤖 Генерация вводного текста резюме с помощью ИИ**：Интеллектуально генерирует профессиональное самоопределение на основе данных резюме (200-300 символов)
  - Поддерживает копирование в буфер обмена, заполнение поля самоопределения, скачивание в виде файла TXT
- Оптимизирует описания работы и проекта по методу STAR
- Автоматически добавляет количественные данные и описания результатов

### 📤 Экспорт в нескольких форматах

- **Экспорт JSON**：Экспортирует полные данные резюме для резервного копирования и синхронизации между устройствами
- **Экспорт LaTeX**：Генерирует профессиональный шаблон резюме в формате LaTeX
  - Может быть напрямую скомпилирован на [Overleaf](https://www.overleaf.com/)
  - Поддерживает китайский язык (с использованием пакета ctex)
  - Профессиональная верстка, подходящая для академических и технических вакансий
  - Включает полные определения стилей и комментарии
- **🤖 Генерация вводного текста резюме с помощью ИИ**：Вызывает модель ИИ для интеллектуальной генерации профессионального самоопределения
  - Генерируется на основе образовательного фона, опыта работы, проектного опыта и навыков из резюме
  - Поддерживает копирование в буфер обмена
  - Поддерживает одноэтапное заполнение поля самоопределения
  - Поддерживает скачивание в виде файла `.txt`
- **Экспорт подсказок для вводного текста резюме**：Экспортирует структурированные шаблоны подсказок, поддерживая `.md`/`.txt`
  - Включает подсказки для личной основной информации
  - Шаблоны вопросов о рабочем опыте
  - Фреймворк для исследования проектного опыта
  - Слова-guides для оценки навыков

### 💾 Постоянное хранение данных

- **Chrome Storage API**：Использует собственный хранилище браузера, данные безопасны и надежны
- **Автоматическое сохранение в реальном времени**：Автоматически сохраняет при изменении содержимого формы, предотвращая потерю данных
- **Ручное сохранение**：Поддерживает ручной клик по кнопке сохранения для подтверждения сохранения
- **Сброс данных**：Одноэтапная очистка всех данных резюме, начало сначала
- **Автоматическое сохранение настроек**：Настройки страницы настроек автоматически сохраняются

## 📦 Руководство по установке

### Метод 1：Установка в режиме разработчика

1. **Скачайте проект**

   ```bash
   git clone https://github.com/itxaiohanglover/offer-laolao-plugin.git
   ```

   Или напрямую скачайте ZIP и извлеките его

2. **Откройте страницу управления расширениями Chrome**

   - Введите в адресную строку：`chrome://extensions/`
   - Или через меню：Больше инструментов → Расширения

3. **Включите режим разработчика**

   - Нажмите переключатель "Режим разработчика" в правом верхнем углу

4. **Загрузите расширение**

   - Нажмите "Загрузить распакованное расширение"
   - Выберите корневой каталог проекта (папку, содержащую `manifest.json`)

5. **Установка завершена**
   - Иконка расширения появится в панели инструментов браузера
   - Нажмите на иконку, чтобы открыть помощник по заполнению резюме

### Метод 2：Установка в браузере Edge

Браузер Edge также поддерживает расширения Chrome：

1. Откройте `edge://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите каталог проекта

## 🚀 Руководство по использованию

### Шаг 1：Настройте API (необязательно, но рекомендуется)

1. Нажмите на иконку расширения, перейдите на вкладку "⚙️ Настройки"
2. **Настройка модели ИИ** (для оптимизации содержимого, рекомендуется)
   - Выберите провайдера модели (например, DeepSeek, Kimi и т.д.)
   - Введите соответствующий API Key
   - Нажмите кнопку "🔗 Проверить подключение" для проверки настроек
3. **Настройка API анализа резюме** (для анализа резюме в формате PDF/DOCX)
   - Купите услугу анализа резюме на [Marketplace Alibaba Cloud](https://market.aliyun.com/detail/cmapi034316)
   - Введите URL API и APP Code

### Шаг 2：Заполните или импортируйте резюме

#### Метод A：Интеллектуальная загрузка и анализ

1. Найдите область загрузки в верхней части вкладки "📝 Заполнение резюме"
2. Перетащите файл резюме или нажмите, чтобы выбрать файл
3. Дождитесь завершения анализа, нажмите кнопку "Использовать данные анализа"
4. Информация о резюме автоматически заполнит форму

#### Метод B：Ручное заполнение

1. Заполните все необходимые данные напрямую в форме
2. Нажмите кнопку "+ Добавить", чтобы добавить несколько записей опыта
3. Данные автоматически сохраняются, или вы можете нажать кнопку "💾 Сохранить" для ручного сохранения

#### Метод C：Импорт JSON

1. Если у вас есть ранее экспортированный файл JSON
2. Просто перетащите его в область загрузки для импорта

### Шаг 3：Оптимизируйте резюме с помощью ИИ (необязательно)

1. Убедитесь, что API Key модели ИИ настроен
2. Заполните описательные содержимое резюме (самоопределение, описание работы, описание проекта и т.д.)
3. Нажмите кнопку "✨ Оптимизировать с помощью ИИ"
4. Система оптимизирует все описательные содержимое по одному
5. Оптимизированное содержимое автоматически заполнит форму обратно

### Шаг 4：Заполните резюме на сайте по поиску работы

#### Одноэтапное предзаполнение (рекомендуется)

1. Откройте страницу заполнения резюме на целевом сайте по поиску работы
2. Нажмите на иконку расширения, чтобы открыть всплывающее окно
3. Нажмите кнопку "📋 Предзаполнить"
4. Расширение автоматически распознает и заполнит поля формы
5. После заполнения будут отображены детали заполнения

#### Точное заполнение на уровне полей

1. Найдите поле, которое вы хотите заполнить, нажмите кнопку "↗" рядом с ним
2. Всплывающее окно автоматически закроется, а страница войдет в "режим заполнения указанием"
3. Нажмите на целевой ввод на веб-странице
4. Значение поля будет точно заполнено
5. Нажмите `Esc` для отмены操作

### Шаг 5：Экспорт и резервное копирование

1. Нажмите кнопку "📤 Экспортировать"
2. Выберите формат экспорта：
   - **JSON**：Для резервного копирования и импорта данных
   - **LaTeX**：Генерирует профессиональный документ резюме, который можно редактировать и печатать на Overleaf
   - **🤖 Генерация вводного текста резюме с помощью ИИ**：Вызывает ИИ для интеллектуальной генерации профессионального самоопределения
     - После генерации вы можете скопировать его, заполнить в самоопределение или скачать в виде файла TXT
   - **Подсказки для вводного текста резюме**：Экспортирует структурированные подсказки для взаимодействия с ИИ (поддерживая `.md`/`.txt`), имя файла автоматически названо "Имя пользователя_Подсказки_Резюме_Дата"

## 🛠️ Технологический стек

- **Фронтенд-фреймворк**：Нативный JavaScript (ES6+)
- **Стандарт расширения**：Chrome Extensions Manifest V3
- **Решение для хранения**：Chrome Storage API + localStorage
- **Решение для стилей**：Нативный CSS (CSS переменные, Flexbox, Grid)
- **Формат документа**：LaTeX (поддержка китайского языка ctex)
- **Интеграция API**：
  - API анализа резюме Alibaba Cloud
  - API больших моделей совместимого с OpenAI формата (DeepSeek, Doubao (Volcano Engine), Tongyi Qianwen и т.д.)

## ⚠️ Важные замечания

1. **Настройка API**：Функция анализа резюме требует настройки API, иначе доступно только импорт JSON
2. **Оптимизация ИИ**：Для использования функции оптимизации необходимо настроить API Key модели ИИ
3. **Совместимость сайтов**：Некоторые сайты могут использовать специальные компоненты форм, поэтому рекомендуется использовать заполнение на уровне полей
4. **Безопасность данных**：Все данные хранятся только локально в браузере и не загружаются на какой-либо сервер
5. **Разрешения браузера**：Расширению требуются разрешения `activeTab`, `scripting` и `storage` для корректной работы
6. **Специальные страницы**：Системные страницы, такие как `chrome://`, `edge://`, `about:`, не поддерживают внедрение скриптов контента
