# 🚀 Offer Laolao - 履歴書自動入力アシスタントスマートブラウザプラグイン

> スマートな履歴書解析と手動入力のデュアルモードをサポートする強力な Chrome ブラウザ拡張機能で、**AI スマートフィールドマッチング**と**フィールドレベルの高精度入力**機能を提供し、求職者が主要な求人サイトで迅速かつ正確に履歴書を記入できるよう支援します。

![Version](https://img.shields.io/badge/Version-1.0-blue)
![Chrome](https://img.shields.io/badge/Chrome-Extension-green)
![Manifest](https://img.shields.io/badge/Manifest-V3-orange)

🌐 **オンラインアクセス**：[https://offer-laolao-plugin.vercel.app](https://offer-laolao-plugin.vercel.app)

## ✨ 主な機能特性

### 📄 スマート履歴書解析

- **多形式サポート**：PDF、DOCX、DOC、TXT、JSON などの主流履歴書形式をサポート
- **クラウド API 解析**：Alibaba Cloud マーケットプレイスの履歴書解析 API を統合し、履歴書情報を正確に抽出
- **JSON 直接インポート**：JSON 形式の履歴書データを直接インポートし、データバックアップと復元を実現
- **ドラッグアンドドロップアップロード**：ファイルのドラッグアンドドロップアップロードをサポートし、操作が簡便
- **スマートフィールドマッピング**：解析結果を自動的に対応するフォームフィールドにマッピング

### 📝 完全な履歴書情報管理

以下の履歴書モジュールの記入と管理をサポート：

| モジュール              | 含まれるフィールド                                                                     |
| ----------------- | ---------------------------------------------------------------------------- |
| **基本情報**      | 氏名、性別、生年月日、携帯電話番号、メールアドレス、身分証明書番号、所在地、政治的面貌         |
| **求職希望**      | 希望職種、希望業界、希望給与、希望勤務地、インターン期間、勤務可能時間                 |
| **教育経験**      | 学校名、専攻、学歴（専門/学士/修士/博士）、ランキング、入学/卒業年月（複数登録可） |
| **職務/インターン経験** | 会社名、職位、開始/終了年月、業務内容（複数登録可）                          |
| **プロジェクト経験**      | プロジェクト名、担当役割、プロジェクト期間、プロジェクト説明、責任内容（複数登録可）                 |
| **スキル情報**      | スキル名、スキルレベル（初級/中級/上級/エキスパート）（複数登録可）                        |
| **言語能力**      | 言語名、習得レベル（入門/基礎/熟練/精通）、言語資格（複数登録可）              |
| **カスタムフィールド**    | カスタムフィールド名と内容（複数登録可）                                             |
| **自己紹介**      | 個人の強みと特徴の展示                                                           |

### 🎯 スマートフォーム入力

#### ワンクリック事前入力機能

- 「📋 事前入力」ボタンをクリックして、履歴書データを現在の求人サイトのフォームに自動入力
- フォームフィールドをスマートに認識し、対応する履歴書情報を自動的にマッチング
- input、textarea、select、contenteditable などの多様なフォーム要素をサポート
- フォームイベント（input、change、blur）を自動的にトリガーし、サイトの検証を確保
- 入力完了後に視覚的なフィードバックを提供し、入力済みのフィールドを強調表示

#### フィールドレベルの高精度入力（↗ ポイント入力）

- 各フィールドに「↗」ボタンがあり、**単一フィールドの高精度入力**をサポート
- ボタンをクリックすると「ポイント入力モード」に入ります：
  - ページ上部に操作ガイドが表示されます
  - マウスが入力可能な要素にホバーすると、ハイライト表示（青い枠）がされます
  - ターゲット入力ボックスをクリックすると、そのフィールド値が入力されます
  - `Esc` キーを押して操作をキャンセルできます
- input、textarea、select、contenteditable 要素への入力をサポート
- フォームイベント（input、change、blur）を自動的にトリガーし、サイトの検証を確保
- 入力成功後にポップアップが自動的に閉じ、連続操作が容易

### 🤖 AI 大規模モデル統合

履歴書コンテンツのスマート最適化とフィールドマッチングに、複数の国内大手モデルサービスプロバイダーをサポート：

| サービスプロバイダー                | サポートモデル                                               |
| --------------------- | ------------------------------------------------------ |
| **DeepSeek**          | DeepSeek Chat、DeepSeek Coder                          |
| **Kimi (Moonshot)**   | Moonshot 8K/32K/128K                                   |
| **Tongyi Qianwen (Alibaba Cloud)** | Qwen Turbo/Plus/Max/Max 長テキスト                         |
| **Volcano Engine (Doubao)**   | Doubao Seed 1.6、Doubao Seed 1.6 Lite、Doubao Seed 1.6 Flash |
| **Zhipu AI**           | GLM-4、GLM-4 Flash、GLM-3 Turbo                        |
| **Baichuan Intelligence**          | Baichuan 2 Turbo、Baichuan 2 Turbo 192K                        |
| **カスタム**            | 任意の OpenAI 互換形式の API をサポート                         |

**AI 機能特性**：

- ワンクリック API 接続テスト
- **✨ AI ワンクリック履歴書最適化**：自己紹介、業務内容、プロジェクト説明などのコンテンツをスマートに最適化
- **🤖 AI 生成履歴書紹介**：履歴書データに基づいてプロフェッショナルな自己紹介をスマートに生成（200-300 文字）
  - クリップボードへのコピー、自己紹介フィールドへの入力、TXT ファイルとしてのダウンロードをサポート
- STAR 法則を使用して業務とプロジェクトの説明を最適化
- 定量的データと成果の説明を自動的に追加

### 📤 複数形式エクスポート

- **JSON エクスポート**：完全な履歴書データをエクスポートし、バックアップとクロスデバイス同期に使用
- **LaTeX エクスポート**：プロフェッショナルな LaTeX 履歴書テンプレートを生成
  - [Overleaf](https://www.overleaf.com/) で直接コンパイル可能
  - 中国語対応（ctex パッケージ使用）
  - プロフェッショナルな排版で、学術的および技術的な求人に適しています
  - 完全なスタイル定義とコメントが含まれています
- **🤖 AI 生成履歴書紹介**：AI モデルを呼び出してプロフェッショナルな自己紹介をスマートに生成
  - 履歴書の教育背景、職務経験、プロジェクト経験、スキル特徴に基づいて生成
  - クリップボードへのコピーをサポート
  - 自己紹介フィールドへのワンクリック入力をサポート
  - `.txt` ファイルとしてのダウンロードをサポート
- **履歴書紹介プロンプトエクスポート**：構造化されたプロンプトテンプレートをエクスポート、`.md`/`.txt` をサポート
  - 個人の基本情報プロンプトを含む
  - 職務経験質問テンプレート
  - プロジェクト経験照会フレームワーク
  - スキル評価ガイダンスワード

### 💾 データ永続化

- **Chrome Storage API**：ブラウザのネイティブストレージを使用し、データは安全で信頼できます
- **リアルタイム自動保存**：フォームコンテンツが変更されると自動的に保存され、データの損失を防止
- **手動保存**：保存ボタンを手動でクリックして保存を確認できます
- **データリセット**：ワンクリックですべての履歴書データをクリアし、再開
- **設定の自動保存**：設定ページの設定は自動的に保存されます

## 📦 インストールガイド

### 方法 1：開発者モードでのインストール

1. **プロジェクトをダウンロード**

   ```bash
   git clone https://github.com/itxaiohanglover/offer-laolao-plugin.git
   ```

   または直接 ZIP をダウンロードして解凍

2. **Chrome 拡張機能管理ページを開く**

   - アドレスバーに入力：`chrome://extensions/`
   - またはメニュー：その他のツール → 拡張機能

3. **開発者モードを有効にする**

   - 右上隅の「開発者モード」スイッチをクリック

4. **拡張機能を読み込む**

   - 「展開済みの拡張機能を読み込む」をクリック
   - プロジェクトのルートディレクトリ（`manifest.json` が含まれるフォルダ）を選択

5. **インストール完了**
   - 拡張機能アイコンがブラウザのツールバーに表示されます
   - アイコンをクリックして履歴書入力アシスタントを開く

### 方法 2：Edge ブラウザでのインストール

Edge ブラウザも Chrome 拡張機能をサポートしています：

1. `edge://extensions/` を開く
2. 「開発者モード」を有効にする
3. 「展開済みの拡張機能を読み込む」をクリック
4. プロジェクトディレクトリを選択

## 🚀 使用チュートリアル

### ステップ 1：API の設定（オプションですが推奨）

1. 拡張機能アイコンをクリックし、「⚙️ 設定」タブに切り替えます
2. **AI モデルの設定**（コンテンツ最適化用、推奨）
   - モデルプロバイダーを選択（DeepSeek、Kimi など）
   - 対応する API Key を入力
   - 「🔗 接続テスト」ボタンをクリックして設定を確認
3. **履歴書解析 API の設定**（PDF/DOCX 形式の履歴書を解析するため）
   - [Alibaba Cloud Marketplace](https://market.aliyun.com/detail/cmapi034316) で履歴書解析サービスを購入
   - API URL と APP Code を入力

### ステップ 2：履歴書の記入またはインポート

#### 方法 A：スマートアップロード解析

1. 「📝 履歴書記入」タブの上部にあるアップロードエリアを見つけます
2. 履歴書ファイルをドラッグアンドドロップするか、ファイルを選択します
3. 解析完了を待ち、「解析データを使用」ボタンをクリックします
4. 履歴書情報がフォームに自動的に入力されます

#### 方法 B：手動記入

1. フォームに直接各種情報を記入します
2. 「+ 追加」ボタンをクリックして複数の経験を追加できます
3. データは自動的に保存されますが、「💾 保存」をクリックして手動で保存することもできます

#### 方法 C：JSON をインポート

1. 以前にエクスポートした JSON ファイルがある場合
2. アップロードエリアに直接ドラッグアンドドロップしてインポートできます

### ステップ 3：AI で履歴書を最適化（オプション）

1. AI モデルの API Key が設定されていることを確認します
2. 履歴書の説明コンテンツ（自己紹介、業務内容、プロジェクト説明など）を記入します
3. 「✨ AI 最適化」ボタンをクリックします
4. システムはすべての説明コンテンツを一つずつ最適化します
5. 最適化が完了すると、コンテンツがフォームに自動的に入力されます

### ステップ 4：求人サイトで履歴書を入力

#### ワンクリック事前入力（推奨）

1. 対象の求人サイトの履歴書記入ページを開きます
2. 拡張機能アイコンをクリックしてポップアップを開きます
3. 「📋 事前入力」ボタンをクリックします
4. 拡張機能が自動的にフォームフィールドを認識して入力します
5. 入力完了後に入力の詳細が表示されます

#### フィールドレベルの高精度入力

1. 入力したいフィールドを見つけ、横にある「↗」ボタンをクリックします
2. ポップアップが自動的に閉じ、ページが「ポイント入力モード」に入ります
3. ウェブページでターゲット入力ボックスをクリックします
4. フィールド値が高精度に入力されます
5. `Esc` キーを押して操作をキャンセルできます

### ステップ 5：エクスポートとバックアップ

1. 「📤 エクスポート」ボタンをクリックします
2. エクスポート形式を選択：
   - **JSON**：データのバックアップとインポートに使用
   - **LaTeX**：Overleaf で編集・印刷できるプロフェッショナルな履歴書文書を生成
   - **🤖 AI 生成履歴書紹介**：AI を呼び出してプロフェッショナルな自己紹介をスマートに生成
     - 生成後はコピー、自己紹介に入力、または TXT ファイルとしてダウンロードできます
   - **履歴書紹介プロンプト**：AI と対話するための構造化プロンプトをエクスポート（`.md`/`.txt` をサポート）、ファイル名は自動的に「ユーザー名_履歴書プロンプト_日付」と命名

## 🛠️ 技術スタック

- **フロントエンドフレームワーク**：ネイティブ JavaScript (ES6+)
- **拡張機能標準**：Chrome Extensions Manifest V3
- **ストレージソリューション**：Chrome Storage API + localStorage
- **スタイルソリューション**：ネイティブ CSS（CSS 変数、Flexbox、Grid）
- **ドキュメント形式**：LaTeX（ctex 中国語サポート）
- **API 統合**：
  - Alibaba Cloud 履歴書解析 API
  - OpenAI 互換形式の大規模モデル API（DeepSeek、豆包（火山引擎）、通义千问など）

## ⚠️ 注意事項

1. **API 設定**：履歴書解析機能には API の設定が必要で、そうでない場合は JSON インポートのみ使用できます
2. **AI 最適化**：最適化機能を使用するには、AI モデルの API Key を設定する必要があります
3. **サイトの互換性**：一部のサイトでは特殊なフォームコンポーネントが使用されている場合がありますので、フィールドレベルの入力を使用することを推奨します
4. **データセキュリティ**：すべてのデータはブラウザのローカルにのみ保存され、どのサーバーにもアップロードされません
5. **ブラウザの権限**：拡張機能は正常に動作するために `activeTab`、`scripting`、`storage` 権限が必要です
6. **特殊ページ**：`chrome://`、`edge://`、`about:` などのシステムページではコンテンツスクリプトの注入はサポートされていません
