# 隐私策略 | Privacy Policy

**最后更新日期：2024年12月**

## 中文版本

### 概述
Offer 捞捞（以下简称"本扩展"）是一款浏览器扩展程序，旨在帮助用户自动填充招聘网站的简历表单。我们非常重视您的隐私。

### 数据收集
**本扩展不收集任何个人数据。**

- 所有简历数据仅存储在您的浏览器本地存储中
- 我们不会将您的数据上传到任何服务器
- 我们不会与第三方共享您的任何信息

### 数据存储
- 您输入的简历信息存储在浏览器的本地存储（Chrome Storage API）中
- 数据完全由您控制，您可以随时删除
- 卸载扩展将自动删除所有存储的数据

### 网络请求
本扩展可能会进行以下网络请求：
- 简历解析 API（仅当您主动上传简历时）
- AI 模型 API（仅当您配置并使用 AI 功能时）

这些请求仅在您主动操作时发生，且不会收集或存储您的个人信息。

### 权限说明
本扩展请求以下权限：
- `storage`：用于在本地存储您的简历数据
- `activeTab`：用于在当前页面填充表单
- `scripting`：用于执行表单填充脚本

### 联系我们
如有任何隐私相关问题，请通过以下方式联系：
- GitHub Issues: https://github.com/itxaiohanglover/offer-laolao-plugin/issues

## English Version

### Overview
Offer Laolao (hereinafter referred to as "this extension") is a browser extension designed to help users auto-fill resume forms on recruitment websites. We take your privacy very seriously.

### Data Collection
**This extension does not collect any personal data.**

- All resume data is stored only in your browser's local storage
- We do not upload your data to any server
- We do not share any of your information with third parties

### Data Storage
- Resume information you enter is stored in the browser's local storage (Chrome Storage API)
- Data is entirely under your control and can be deleted at any time
- Uninstalling the extension will automatically delete all stored data

### Network Requests
This extension may make the following network requests:
- Resume parsing API (only when you actively upload a resume)
- AI model API (only when you configure and use AI features)

These requests only occur when you actively initiate them and do not collect or store your personal information.

### Permissions
This extension requests the following permissions:
- `storage`: To store your resume data locally
- `activeTab`: To fill forms on the current page
- `scripting`: To execute form-filling scripts

### Contact Us
For any privacy-related questions, please contact us via:
- GitHub Issues: https://github.com/itxaiohanglover/offer-laolao-plugin/issues
